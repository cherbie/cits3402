#### Usage
Use the Open MPI compiler wrapper to compile the source code files. A *makefile* has been provided.

```
    @user~: make
```

To run the application use the Open MPI command line utility *mpirun*

```
    @user~: mpirun ./sp [-f] filename.in
```
